# Networking Basics for Cybersecurity

Este repositorio contiene mis apuntes y prácticas basados en el curso de Cisco Networking Academy: Networking Basics.

El objetivo es construir una base sólida en redes, entendiendo cómo funcionan las infraestructuras de red y su aplicación en ciberseguridad.

## 🎯 Objetivo

Desarrollar conocimientos fundamentales de redes aplicados a escenarios de ciberseguridad, incluyendo análisis de tráfico, identificación de servicios y comprensión de infraestructuras de red.

---

## 🧠 Sobre el curso

El contenido de este repositorio está basado en el curso *Networking Basics* de Cisco, el cual cubre los fundamentos de redes como:

- Funcionamiento de redes y dispositivos
- Modelo OSI y TCP/IP
- Direccionamiento IP
- Protocolos de red
- Conceptos básicos de seguridad de red

Este tipo de formación proporciona una base esencial para roles técnicos en IT y ciberseguridad. :contentReference[oaicite:0]{index=0}
