# Networking Basics

Este repositorio contiene mis apuntes y resúmenes basados en el curso de Cisco Networking Academy: *Networking Basics*.

El objetivo es construir una base sólida en redes, comprendiendo el funcionamiento de las infraestructuras, protocolos y conceptos fundamentales.

## 🎯 Objetivo

Adquirir una base sólida en redes que permita comprender cómo se comunican los sistemas, cómo se estructuran las redes y cómo funcionan los principales protocolos.

---

## 🧠 Sobre el curso

El contenido de este repositorio está basado en el curso *Networking Basics* de Cisco Networking Academy, el cual cubre conceptos fundamentales como:

- Funcionamiento de redes
- Modelos OSI y TCP/IP
- Direccionamiento IP
- Protocolos de red
- Dispositivos de red
